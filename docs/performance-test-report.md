## 测试场景

### 1. 健康检查 (Health Check)
- **端点**: `GET /health`
- **检查项**:
  - HTTP 状态码 200
  - 响应时间 < 200ms

### 2. 客户端配置 (Client Config)
- **端点**: `GET /api/config/client`
- **检查项**:
  - HTTP 状态码 200
  - 返回数据包含 `data` 字段

### 3. 用户登录 (User Login)
- **端点**: `POST /api/auth/login`
- **检查项**:
  - HTTP 状态码 200
  - 返回 `access_token`

### 4. 管理员配置 (Admin Config)
- **端点**: `GET /api/admin/configs`
- **前置条件**: 登录成功获取 Token
- **检查项**:
  - HTTP 状态码 200
  - 返回配置列表数组

### 5. API 版本 (API Version)
- **端点**: `GET /api/version`
- **检查项**:
  - HTTP 状态码 200
